# Modern Data Warehousing Project

## Objective
- Develop a modern data warehouse using SQL Server to consolidate sales data, enable analytical reporting and inform decision making.
- Develop SQL-based analytics to deliver detailed insights into customer behaviour, product performance and sales trends.

## Functional Specification
- **Sources:** Import data from an ERP and CRM, provided as CSV files (can we make this something else).
- **Data Quality:** Cleanse and resolve DQ issues prior to analysis.
- **Integration:** Combine both sources into a single, user friendly model for analytical queries.
- **Scope:** Focus on latest dataset only - no need for history.
- **Documentation:** Provide clear documentation of data model to support stakeholders and for analytics.
